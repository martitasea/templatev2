# plantilla

Dependencias
---
- React 17.0.0
- Storybook 6.3.12
- Webpack 5.60.0
- MUI 5.1.0

Plantilla para un proyecto con React, MUI y storybook
---
Asegurate de actualizar el nombre del proyecto en los siguientes archivos:
* package.json:
    * name
    * repository.url
    * bugs.url
    * homepage

* package-lock.json:
    * name
* template.html:
    * Etiqueta <title>
